# Fotbollsverkstan-booking
Enkel bokningsapp för öppna träningar i Fotbollsverkstan där deltagare kan boka tider.
## Summary
This project is a booking system for Fotbollsverkstan. 
Users can sign up for training sessions by entering their name, email, and selecting a time slot. 
The system stores bookings in a Google Sheet using Google Apps Script.
